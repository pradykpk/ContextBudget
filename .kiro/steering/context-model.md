# Context Model

## Context Categories

1. System Critical
   - Safety rules
   - Governance constraints
   - Non-removable

2. Task Instructions
   - Current task objectives
   - High priority

3. Decisions & Assumptions
   - Explicit choices
   - Must be preserved

4. Working Context
   - Active reasoning
   - Eligible for summarization

5. Historical Relevant
   - Prior discussions
   - Compressible

6. Background Reference
   - Explanatory content
   - Lowest priority

## Protection Rules
- Categories 1–3 are protected
- Categories 4–6 may be summarized only with approval
